# Proyecto de Traducción Asháninka 🇵🇪

Este repositorio contiene los avances, documentación y herramientas desarrolladas para el proyecto de traducción y preservación de la lengua Asháninka.

## 💻 Desarrollo Técnico
En este repositorio encontrarás los componentes clave del sistema de IA:

* **[`Trabajo_IA_Final.ipynb`](./Trabajo_IA_Final.ipynb):** Cuaderno de Jupyter con el proceso de entrenamiento, preprocesamiento y evaluación de los modelos de traducción. 
    > *Tip: GitHub permite visualizar este archivo directamente en el navegador.*

* **[`ashaninka_hf.jsonl`](./ashaninka_hf.jsonl):** Dataset en formato JSON Lines, estructurado para ser compatible con plataformas como Hugging Face. Contiene los pares de oraciones para el entrenamiento.

* **Bibliotecas:** Transformers, Pandas, Scikit-learn
